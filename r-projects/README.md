# R Projects

Reproducible statistical analysis and data science projects using R and R Markdown.

## Projects

### 1. Food Security Analysis - Mali (FIES)
Analysis of Food Insecurity Experience Scale data for Mali.
- **File**: `food-security-mali/fies_mali_RDATA.Rmd`
- **Methods**: Descriptive statistics, food insecurity indicators
- **Output**: Reproducible R Markdown report

### 2. Principal Component Analysis - Environment
PCA analysis for environmental data.
- **File**: `pca-environment/pca_environnement.Rmd`
- **Methods**: PCA, dimensionality reduction, visualization
- **Output**: Reproducible R Markdown report

## Technical Stack
- R Markdown for reproducible research
- tidyverse for data manipulation
- ggplot2 for visualization
- Statistical modeling and analysis
