# Food Security Analysis - Mali (FIES)

## Description
Analysis of Food Insecurity Experience Scale (FIES) data for Mali. This project examines household food security indicators and patterns.

## Objective
- Analyze food insecurity prevalence in Mali
- Generate descriptive statistics on FIES indicators
- Produce reproducible analysis with R Markdown

## Data Source
FIES (Food Insecurity Experience Scale) data for Mali

## Methods
- Descriptive statistics
- Food security indicators calculation
- Data visualization
- R Markdown for reproducible reporting

## File
- `fies_mali_RDATA.Rmd` - Main analysis script (R Markdown)

## How to Use
```r
# Open in RStudio
# Knit the document to generate HTML/PDF report
# Or run chunks interactively
```

## Output
HTML/PDF report with analysis results, tables, and visualizations

## Requirements
```r
# Install required packages
install.packages(c("tidyverse", "knitr", "rmarkdown"))
```
