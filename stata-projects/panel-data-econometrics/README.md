# Panel Data Econometrics - Technology & Inequality

## Research Question
**How do technological innovations affect income inequality across countries?**

## Description
Complete econometric research project analyzing 30-year panel data (1993-2022) from World Bank WDI to examine the relationship between technological advancement and income distribution.

## Data

**Source**: World Bank World Development Indicators (WDI)  
**Coverage**: 1993-2022 (30 years)
