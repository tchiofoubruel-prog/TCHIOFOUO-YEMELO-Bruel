# Survey Data Analysis - Sampling & Weighting Methods

## Description
Professional handling of complex household survey data with focus on sampling design, probability weights, and representative inference.

## Topics Covered

### 1. Sampling Weights
- Probability Proportional to Size (PPS) sampling
- Household inclusion probabilities
- Weight calculation and validation

### 2. Survey Design
- Primary Sampling Units (PSUs)
- Stratification
- Clustering effects
- Design effects

### 3. Post-Survey Adjustments
- Non-response corrections
- Invalid household adjustments
- Post-stratification by district
