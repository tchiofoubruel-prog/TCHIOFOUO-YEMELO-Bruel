# Stata Projects - Econometrics & Impact Evaluation

Applied econometric analysis and impact evaluation projects using Stata.

## Projects

### 1. Impact Evaluation Methods - Replication Studies
Advanced causal inference methods replicated from published papers.
- **File**: `impact-evaluation-methods/impact_analysis_methods.do`
- **Methods**: RCT, RDD, Difference-in-Differences, PSM, Instrumental Variables
- **Papers replicated**: Mozambique political economy, various impact evaluations
- **Skills**: Treatment effect estimation, balance tests, robustness checks

### 2. Panel Data Econometrics - Technology & Inequality
Complete econometric analysis of technological innovation's effect on income inequality.
- **File**: `panel-data-econometrics/panel_data_analysis.do`
- **Data**: World Bank WDI (1993-2022), 30-year panel
- **Methods**: Fixed effects, random effects, Hausman tests, dynamic panel models
- **Application**: Research report on technology and inequality nexus

### 3. Survey Data Analysis - Sampling & Weighting
Professional survey data handling with complex sampling designs.
- **File**: `survey-data-analysis/work_on_survey_data.do`
- **Topics**: Sampling weights, stratification, PSU analysis, post-stratification
- **Application**: Household survey methodology (Albania LSMS example)
- **Skills**: Survey design, weighting adjustments, representative inference

### 4. Household Survey Analysis - Rwanda LWH
Agricultural household survey data cleaning and analysis.
- **File**: `household-survey-rwanda/LWH_Project_code.do`
- **Data**: Rwanda household follow-up survey (LWH_FUP2)
- **Tasks**: Data cleaning, unit conversion, consumption analysis, crop production
- **Skills**: Missing data handling, outlier treatment, agricultural indicators

### 5. Education RCT Analysis - Subsidy Program
Randomized controlled trial evaluation of education subsidy program.
- **File**: `education-rct-analysis/code.do`
- **Design**: School-level randomization (2010-2012)
- **Outcomes**: School dropout, teen pregnancy, marriage
- **Methods**: Balance tests, treatment effect estimation, heterogeneity analysis
- **Skills**: RCT analysis, clustering, stratification, LaTeX table output
