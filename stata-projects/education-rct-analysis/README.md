# Education RCT Analysis - Subsidy Program Evaluation

## Research Question
**Does an education subsidy program reduce school dropout, teen pregnancy, and early marriage?**

## Study Design

**Type**: Randomized Controlled Trial (School-level randomization)  
**Period**: 2010-2012  
**Target**: Grade 6 students in public schools  
**Treatment**: Education subsidy program  
**Follow-up**: 3 years and 5 years post-treatment
